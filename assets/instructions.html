This api allows you to search and update menus by:

  1. Category
    2. A list of categorys will be displayed if you enter /categorylist
  2. Name
    3. Search for partial name or an exact match
  3. Price
    4. Search for prices within a $2.00 range. e.g. if you enter $4.95 you'll get all products within the $4.95 to $6.95 range.
  4. Id
Searching is open to the world, but updating requires an auth key.
