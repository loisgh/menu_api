This api allows you to search and update menus by:

  1. Category
    2. A list of categorys will be displayed if you enter /categorylist
  2. Name
    3. Search for partial name or an exact match
  3. Price
    4. Search for prices within a $2.00 range. e.g. if you enter $4.95 you'll get all products within the $4.95 to $6.95 range.
  4. Id
Searching is open to the world, but updating requires an auth key.
If you wish to update this database, you must use an api key. You can request a key by contacting me @ loisgh@gmail.com
An add request looks as follows curl --data 'descr=test&name=test&m_type=test&price=1.007&access_token=yourkeyhere' "http://localhost:4567/add/menu_item"
A delete request looks as follows curl --data 'access_token=yourkeyhere' "http://localhost:4567/remove/:id/delete"
An update request looks as follows curl --data 'descr=test&name=test&m_type=test&price=1.007&access_token=yourkeyhere' "http://localhost:4567/update/:id"
